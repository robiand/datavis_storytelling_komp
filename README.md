This repo is used to host my storytelling graphs and EDA-file from the [group project Crypto Dashboard/CoinCompass](https://github.com/DataHub2/crypto-dashboard-DE25-UX25)

The data used for the notebook in this repo is "master_cleaned.csv" generated and transformed by "fetch_data.py" and "transform.py" in said group repo.

Requirements:
- ipykernel
- pandas
- matplotlib

Results in EDA.ipynb should be viewable directly on GitHub, to run the code:

- Install requirements
- Run all cells in notebook
